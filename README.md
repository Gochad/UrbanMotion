# AbstractProgrammingProject
repo for my AbstractProgrammingProject for JU classes

## How to run it
1. `docker build -t cpp_project .`
2. `docker run --rm cpp_project`

## how to run locally
1. `mkdir build`
2. `cd build`
3. copy `map1.txt` to `build` dir
4. `cmake ..`
5. `cmake --build .`
6. `./UrbanMotion`
